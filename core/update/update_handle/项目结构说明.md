# 项目结构说明

## 目录结构

```
update_handle/
├── TCUUpdate.pro          # Qt项目文件
├── main.cpp               # 主程序入口
├── mainwindow.h/cpp       # 主窗口类（GUI）
├── mainwindow.ui          # UI界面文件
├── updateengine.h/cpp     # 更新引擎（核心逻辑）
├── scriptparser.h/cpp     # XML清单解析器
├── backupmanager.h/cpp    # 备份管理器
├── updateexecutor.h/cpp   # 更新执行器
├── configmanager.h/cpp    # 配置管理器
├── logger.h/cpp           # 日志管理器
├── diskspacechecker.h/cpp # 磁盘空间检查器
├── config/                # 配置文件目录
│   ├── tcu_update.conf    # 配置文件示例
│   ├── README.md          # 配置说明
│   └── 配置部署脚本.sh    # 配置部署脚本
└── release/               # 编译输出目录
    ├── bin/               # 可执行文件
    │   └── tcu_update     # 主程序
    └── obj/               # 目标文件（.o文件）
```

## 编译输出说明

### release/bin/

存放编译生成的可执行文件：
- `tcu_update`：主程序可执行文件

### release/obj/

存放编译过程中的中间文件：
- `*.o`：目标文件
- `moc_*.cpp`：Qt元对象编译器生成的文件
- `ui_*.h`：UI文件编译生成的头文件

## 编译配置

在 `TCUUpdate.pro` 中配置了输出路径：

```qmake
DESTDIR = release/bin
OBJECTS_DIR = release/obj
MOC_DIR = release/obj
RCC_DIR = release/obj
UI_DIR = release/obj
```

## 清理编译文件

```bash
# 清理所有编译文件
make clean

# 或手动删除
rm -rf release/
```

## 部署说明

编译完成后，需要部署以下文件到目标设备：

1. **可执行文件**：`release/bin/tcu_update`
2. **配置文件**：`config/tcu_update.conf` -> `/usr/app/config/tcu_update.conf`

部署步骤：
```bash
# 1. 复制可执行文件
cp release/bin/tcu_update /usr/bin/

# 2. 部署配置文件
cd config
./配置部署脚本.sh
```
