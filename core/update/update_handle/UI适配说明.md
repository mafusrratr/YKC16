# UI 适配说明

## 屏幕规格

- **分辨率**：800 x 400 像素（固定）
- **宽高比**：2:1
- **设备类型**：嵌入式触摸屏设备
- **Qt版本**：4.8.6

## UI 设计

### 窗口设置

- **窗口大小**：800 x 400（固定，不可调整）
- **最小尺寸**：800 x 400
- **最大尺寸**：800 x 400
- **全屏模式**：建议使用全屏模式（可后续添加）

### 布局结构

```
总高度: 400px
├── 上边距: 10px
├── 状态标签: ~40px (最大高度，自适应)
├── 间距: 5px
├── 进度条: 25px
├── 间距: 5px
├── 日志区域: ~280px (最小高度，可拉伸)
├── 间距: 5px
├── 按钮区域: 35px
└── 下边距: 10px
```

### 控件规格

| 控件 | 高度 | 字体大小 | 说明 |
|------|------|----------|------|
| 状态标签 | 最大40px | 12px | 支持文本换行 |
| 进度条 | 25px | 自动 | 显示百分比 |
| 日志区域 | 最小280px | 10pt | 等宽字体，自动滚动 |
| 按钮 | 35px | 12px | 触摸操作友好 |

### 字体设置

- **状态标签**：系统默认字体，12px，粗体
- **日志文本**：系统默认字体，10pt，等宽样式
- **按钮**：系统默认字体，12px

### 颜色方案

- **成功状态**：绿色文字
- **失败状态**：红色文字
- **回滚按钮**：红色背景（#ff6b6b），白色文字

## 触摸操作优化

1. **按钮最小高度**：35px（符合触摸操作标准）
2. **进度条高度**：25px（便于查看）
3. **间距**：5px（紧凑但不拥挤）

## 注意事项

1. **固定分辨率**：程序仅适配 800x400，不支持其他分辨率
2. **字体大小**：已优化为适合小屏幕的尺寸
3. **日志显示**：由于高度限制，可能需要滚动查看
4. **触摸操作**：按钮高度35px符合触摸操作标准
